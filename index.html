PROBE-FDAA User Guide

How to use this tool

FDAAs have differential incorporation in different bacteria. Using this fact, we performed a screen of 7 compounds in 17 bacterial species to identify potential patterns of incorporation. You may use the results of this screen in this tool to perform cellular identification. To do this, please enter the bacteria you would like to distinguish in the checkboxes. The systen will then determine which FDAAs may be used to achieve separation. Each number within the table represents the relative Mean Fluorescence Intensity (MFI) from a flow cytometry experiment, greater MFIs represent increased labeling. Rows highligted in red are predicted to be distinguishable based on the activity of that compound. Predicted growth times are also given to allow full incorporation. For any questions, please contact tcwarner@mit.edu.
